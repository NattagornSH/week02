# Project Overview: Japanese Green Tea E-commerce

This project is a specialized e-commerce platform for Japanese green tea (Matcha, Hojicha, Sencha). The goal is to provide customers with high-quality tea leaves and educational content about different tea types, their flavor profiles, and proper brewing techniques.

## Current Project Phase
The project is currently in the **Planning and Prototyping** phase. It includes business modeling, technical diagrams, a product backlog, and an initial JavaScript data structure prototype.

## Key Files
- **01_my-ecommerce.md**: Project problem statement and solution overview.
- **02_business-model-canvas.excalidraw**: Business model and strategy visualization.
- **03_use-case-diagram.excalidraw**: Functional requirements and user interactions.
- **04_er-diagram.excalidraw**: Database schema and entity relationships.
- **05_product-backlog.md**: User stories and feature requirements for customers and staff.
- **06_bring-it-to-javascript.js**: Initial data structure prototype for products and shopping cart.

## Development Status
- **Backend/Database**: Conceptualized in the ER diagram (04); not yet implemented.
- **Frontend**: Not yet started; initial logic explored in `06_bring-it-to-javascript.js`.
- **Technologies**: JavaScript (ES6+), Excalidraw for diagrams.

## How to Use
1. **Research & Planning**: Consult the `.excalidraw` files and `01_my-ecommerce.md` to understand the system's design and goals.
2. **Feature Development**: Refer to `05_product-backlog.md` for the list of features to implement.
3. **Data Prototyping**: Use `06_bring-it-to-javascript.js` as a reference for the initial product and cart object structures.

## Usage Commands
No build system is currently configured. To run the prototype script:
```bash
node 06_bring-it-to-javascript.js
```
